# STL DOM Interface

This is the standard `dom` handling package for LovelyIO

## Usage

Install it locally with `lovely install dom` or hook up from
`http://cdn.lovely.io`, and then

    Lovely(['dom'], function($) {
      $('#my-id').highlight();
    });

See the API docs for the actual usage information
